# Мұғалім Өскемен — Стипендиялық бағдарлама

Болашақ мұғалімдердің жаңа буынын құру арқылы саналы қоғамның қалыптасуына үлес қосатын бағдарламаның веб-сайты.

## Жылжыту

```bash
npm install
npm start
```

Браузерда [http://localhost:3000](http://localhost:3000) ашылады.

## Жинақтау

```bash
npm run build
```

## Жаңартулар (mugalimoskemen-мен салыстырғанда)

- **Дизайн**: Manrope қаріпі, жақсартылған типография, анимациялар
- **UX**: Sticky header, жол ауысқанда жоғарыға скролл, CTA түймелері
- **Мазмұн**: About, Participants, Register, FAQ беттері толық толтырылды
- **Байланыс**: Контактілер карточкалар түрінде, басқаруға ыңғайлы
- **Қолдау**: Банк мәліметтері кесте түрінде, нөмірді көшіру түймесі
- **Бірлестіктер**: Сетка макеті, hover эффектілері
- **Қолжетімділік**: Skip link, aria-label, семантикалық HTML
- **SEO**: Толық meta сипаттамасы, тіл атрибуты (kk)
